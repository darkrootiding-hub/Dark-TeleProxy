📍 Where things are + how to use them
Chat & input
main center + bottom bar
  • Type message, Enter send / Shift+Enter new line
  • Voice: click mic, speak – auto‑sends after pause
  • Prompt suggestions appear when chat is empty – click to fill
  • Code blocks have copy + run (opens IDE)
Sidebar
click ☰ top‑left
  • Recent sessions: click any to resume, trash to delete
  • Quick actions: Study hacking, Feedback, Prompt manager, Activate DarkRoot AI
  • Dynamic buttons (featured actions) – click to insert prompt
  • Auth: login/logout at bottom
IDE (code editor)
header button / from code blocks
  • Languages: HTML, Python, C, C++, JS
  • Gestures: pinch to zoom font, two‑finger scroll
  • Shortcuts: Ctrl+Enter run, Ctrl+S save
  • Console + stdin for Python/C
Legal hacking
sidebar “Study Legal Hacking” or /study
  • Categories: basics, network, web, crypto, forensics, tools
  • Click card → AI switches to ethical teaching mode
  • Always legal/defensive focus
Settings
header button
  • AI model selector (DarkRoot AI default, Gemini, Llama, Mistral…)
  • Theme, music player, response length, custom prompts
  • Clear session memory
Auth & cloud
sidebar bottom + auth modal
  • Login/signup with email, “Remember me” for auto‑login
  • Sessions sync to Firestore after login
  • Guest mode uses localStorage
